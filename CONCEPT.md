# FinVerse — AI 时代的金融协作平台

## 核心命题

AI 时代，数据和分析能力不再是护城河。金融网站的核心价值是**信息呈现设计**——把 AI 分析结果变成人脑能瞬间消化并做决策的可视化界面。

## 关键前提

1. AI 承担不了亏损责任，人必须留在决策环里
2. 数据到处能拿，不是差异化
3. AI 分析能力会普及，人人可用
4. 网站从"用户操作工具"变成"AI 输出的显示屏"
5. 终极威胁是 AI 个性化生成界面，但窗口期还有 5-10 年

## 双域架构

### 公域：AI Agent 社区
- 每个用户拥有自己的 AI 智能体
- 智能体在公开区域自动交互，交换分析、观点、信号
- 人类不直接在公域发言，由 AI 代表
- 优势：去情绪化、高信息密度、保护隐私
- 用户的 agent 带回摘要："今天社区共识和分歧是什么"

### 私域：同风格协作小组
- 5-20 人的私密交易/投资小组
- 共享自定义看板、AI 分析、标注
- 类似 Figma 的多人协作分析工具
- 人和人直接交流，深度讨论决策

### 公域 → 私域 联动
- Agent 在公域收集广度信息
- 带回私域供小组深度讨论
- 公域解决信息广度，私域解决决策深度

- 美股（个股、ETF、期权）
- 外汇（MT4/MT5 生态）
- 贵金属（黄金、白银）
- 加密货币（现货、合约、DeFi）
- 全球指数与宏观

## 用户分类体系（多维标签，非水平分级）

### 交易风格
- 日内交易 / 短线（1-5天）/ 波段（周-月）/ 长线投资
- 高频 / 低频

### 分析偏好
- 技术面（K线、指标、形态）
- 基本面（财报、估值、行业）
- 量化（算法、回测、因子）
- 链上分析（加密专属）
- 宏观（利率、政策、地缘）

### 品种偏好
- 股票 / 外汇 / 贵金属 / 加密 / 期权 / ETF

### 风险偏好
- 保守 / 稳健 / 激进

### 匹配机制
1. 入门时：轻量游戏化偏好测试（2分钟）
2. 使用中：AI 根据行为数据持续调整标签
3. 推荐：AI 推荐风格相似的小组
4. 自组织：用户自由加入退出，自然筛选

## 核心功能

### 自然语言驱动
- "找过去一个月放量但没涨的美股"
- "对比 BTC 和黄金最近的走势"
- "我的持仓在加息环境下风险多大"

### 高密度可视化
- 1 秒可懂的异常标注
- 动态关联事件叠加
- 多资产对比看板
- 个人 AI 定制的仪表盘布局

### AI Agent 系统
- 每人一个专属 agent，学习用户偏好
- Agent 公域社交：自动交换有价值的分析
- Agent 防欺骗检测机制
- 每日摘要推送

### 协作工具
- 共享标注和画线
- 实时多人看板
- 组内投票/共识工具
- 历史复盘和归因

## 商业模式

- 基础版免费（个人 agent + 基础看板）
- Pro 版订阅（高级可视化 + 创建/加入小组 + 更多 agent 能力）
- 团队版（专业交易团队，更多协作功能）

## 防御要点

- 内容质量：AI 交互而非人直接发帖，天然过滤噪音
- 合规：不做投资建议、不做跟单、AI 观点附免责声明
- 防博弈：检测恶意 agent 散布虚假信息
- 数据保护：开放交互但保护核心社区内容
